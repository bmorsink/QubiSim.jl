# Learning by Example

Learning through examples is a practical way to build skills and connect theory to practice. QubiSim supports this by offering a diverse collection of examples, covering the following aspects of quantum algorithms, protocols and experiments:

## Algorithms

- Quantum phase estimation
- Quantum counting
- Grovers algorithm
- Bernstein Vazirani algorithm
- Simon's algorithm

## Protocols

- Superdense coding
- Quantum teleportation
- Quantum key distribution

## Experiments

- CHSH game
- Bell state experiments
- Quantum no-signaling theorem
- Quantum erasure experiments
- Reduced density operators
- Quantum parallellism
