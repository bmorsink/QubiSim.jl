# Concepts, workflow and core components 

A quantum algorithm simulation in QubiSim can be decomposed into four main stages. Each stage corresponds to a specific transformation of the system, supported by dedicated data structures and functions.

- [Quantum Circuits](guide_circuits.md) — how you define the quantum circuit
- [Compilation](guide_compilation.md) — how you compile the quantum circuit into a quantum program 
- [Execution](guide_execution.md) — how you execute the quantum program on quantum states
- [Results](guide_results.md) — how you process the results
